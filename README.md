# mishka

Проект «Мишка» от HTML Academy

[Опубликованная версия](https://efiand.github.io/mishka/).

## Инструкция по применению

### Установка модулей

```bash
npm ci
```

### Сборка проекта в папку build

```bash
npm run build
```

### Запуск в режиме разработки

```bash
npm run dev
```

### Запуск линтеров и валидаторов

```bash
npm run lint
```
